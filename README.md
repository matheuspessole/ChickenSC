# ChickenSC - Diagnostic Tool (v3.0)

![Status](https://img.shields.io/badge/Status-Open%20Source-blue) 
![Language](https://img.shields.io/badge/Language-C-blue) 
![License](https://img.shields.io/badge/License-Non--Commercial-orange)

High-performance diagnostic utility for hardware auditing. **Version 3.0** marks a complete architectural shift: we have transitioned from the previous Python/Lua-based structure to a native implementation in **C**. This overhaul ensures native execution, minimal resource consumption, and the total elimination of external dependencies. Fully optimized for **x86 and x64** architectures, with complete backward compatibility.

### ✨ Key Features (v3.0)

* **Native Performance:** Single binary compiled in C. We have eliminated the need for interpreters (Python) or additional runtimes.
* **Zero Dependencies:** Direct installation and execution. No `pip`, virtual environments, or external libraries required.
* **Extreme Efficiency:** Instant processing with highly optimized memory and CPU usage.
* **Universal Compatibility:** Full support from Windows XP through modern versions.

---

### 🚀 How to Use

#### System Requirements
* **Operating System:** Windows XP, 7, 10, or 11 (x86 or x64).
* **Installation:** No environment configuration required. Simply download the latest version from the [Releases](https://github.com/matheuspessole/ChickenSC/releases) section and run the binary.

#### Security Notice
ChickenSC is an open-source project without a proprietary digital certificate. Due to this, the operating system may impose preventive restrictions during execution or removal.
