# ChickenSC - Ferramenta de Diagnóstico (v3.0)

![Status](https://img.shields.io/badge/Status-Código%20Aberto-blue) 
![Language](https://img.shields.io/badge/Language-C-blue) 
![License](https://img.shields.io/badge/License-Não--Comercial-orange)

Utilitário de diagnóstico de alta performance para auditoria de hardware. A **versão 3.0** marca uma mudança completa na arquitetura do projeto: abandonamos a estrutura baseada em Python/Lua para uma implementação integral em **C**. Esta transição garante execução nativa, consumo mínimo de recursos e eliminação total de dependências externas. Totalmente otimizado para arquiteturas **x86 e x64**, com suporte retroativo total.

### ✨ Principais Recursos (v3.0)

* **Performance Nativa:** Binário único compilado em C. Eliminamos a necessidade de interpretadores (Python) ou runtimes adicionais.
* **Zero Dependências:** Instalação e execução direta. Nada de `pip`, ambientes virtuais ou bibliotecas externas.
* **Eficiência Extrema:** Processamento instantâneo com uso otimizado de memória e CPU.
* **Compatibilidade Universal:** Suporte completo desde o Windows XP até versões modernas.

---

### 🚀 Como Usar

#### Requisitos de Sistema
* **Sistema Operacional:** Windows XP, 7, 10 ou 11 (x86 ou x64).
* **Instalação:** Não requer configurações de ambiente. Basta baixar a versão mais recente na seção [Releases](https://github.com/matheuspessole/ChickenSC/releases) e executar o binário.

#### Aviso de Segurança
O ChickenSC é um projeto de código aberto sem certificação digital proprietária. Devido a esta característica, o sistema operacional pode impor restrições preventivas durante a execução ou remoção. 

---

### ⚠️ Informações sobre a Desinstalação (PT-BR/EN)

**PT-BR:** Por ser um software de código aberto sem certificação digital, o Windows pode restringir a desinstalação automática. Caso ocorra erro, localize a pasta do programa, execute o arquivo "unins000.exe" ou exclua o diretório manualmente para concluir a remoção.

**EN:** As an open-source software without digital certification, Windows may restrict automatic uninstallation. If an error occurs, locate the program folder, run the "unins000.exe" file, or delete the directory manually to complete the removal process.
